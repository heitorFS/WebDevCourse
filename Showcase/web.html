Funcionamento da web
A internet funciona como a imagem sugere! Ao entrer numa página na web, você realiza uma "pergunta" ao seu provedor de internet (ISP): "Quem é esse google.com?". O provedor realiza essa pergunta ao DNS, que seria uma espécie de lista telefônica da internet, possuindo todos os websites e seus respectivos endereços. Retornando assim, o IP do site desejado, e nesse IP, os servidores do website retorna para o usuário 3 arquivos: HTML, CSS, JS. Finalmente permitindo com que o website seja usado como você quiser!
Backbone da Internet
O "Backbone" da internet é o trecho de comunicação do ISP com os servidores do mundo todo para acessarmos websites, ou seja, o backbone é uma imensa conexão via cabo de países (como podemos ver no website https://www.submarinecablemap.com) para o tráfego de dados por todo o mundo.
