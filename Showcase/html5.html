HTML Básico
O HTML é acompanhado de 5 tags principais em todos seus códigos, sendo elas:
<!DOCTYPE html><!-- Isto é um comentário! DOCTYPE é responsável por esclarecer ao navegador qual é o tipo de documento a ser renderizado -->
<html><!-- Esta tag é responsável por declarar que o código que virá a seguir será um código html -->
 <head>
  <title>Meu primeiro website</title><!-- Esta tag é responsável por dar um título à aba no navegador -->
 </head>
 <body><!-- Esta tag será responsável pela parte visível de seu website -->
  Corpo do meu primeiro site... Olá, pessoal
 </body>
</html>

  • <h1> - <h6>: Títulos, diminuindo o tamanho de acordo com o número de forma descrescente.
  • <p>: Parágrafo, quebrando a linha quando a tag é fechada.
  • <strong>: Negrito.
  • <em>: Títulos, diminuindo o tamanho de acordo com o número de forma descrescente.

HTML Avançado
Formulários são uma das partes mais importante para um Dev HTML
<!DOCTYPE html>
<html>
 <head>
  <title>Registrar</title>
 </head>
 <body>
  <form method="get">
   Nome: <input type="text" name="nome" /><br />
   Sobrenome: <input type="text" name="sobrenome" /><br />
   Email: <input type="email" required name="email" /><br />
   Senha: <input type="password" min="4" name="senha" /><br />
   Nascimento: <input type="date" name="nascimento" /><br />
   Sexo: <br />
   <input type="radio" value="masculino" name="sexo" />Masculino<br />
   <input type="radio" value="feminino" name="sexo" />Feminino<br />
   Pets: <br />
   <input type="checkbox" name="gato" />Gato<br />
   <input type="checkbox" name="cachorro" />Cachorro<br />
   Carros: <br />
   <select name="carros">
    <option value="volvo">Volvo</option>
    <option value="audi">Audi</option>
   </select><br />
   <input type="submit" value="Registrar" />
   <input type="reset" value="Limpar" />
  </form>
 </body>
</html>
