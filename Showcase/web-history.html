WWW
Tim Berners-Lee foi o primeiro desenvolvedor para web. A internet era uma rede mundial, entretanto, diversos computadores usavam protocolos distintos, então era quase impossível uma precisa conexão com o resto dos servidores. Berners-Lee foi responsável pela criação do WWW, em 1991, World Wide Web, um protocolo padrão para que o mundo todo pudesse se comunicar num mesmo padrão.
HTML, CSS, JS
HTML é uma forma para escrevermos texto num website, entretanto, em 1995 foram criadas duas ferramentas que revolucionariam nosso conhecimento sobre o funcionamento de websites. O CSS, responsável pela parte visual do website, e o Javascript seria uma programação no website com o objetivo de tornar o site dinâmico.
